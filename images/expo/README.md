# Expo Images Directory

Place your expo images in this directory with the following naming convention:

## Required Image Files:
- `smartbharat-2024.jpg` - SmartBharat Expo 2024 image
- `electronica-india-2023.jpg` - Electronica India & Productronica India 2023 image  
- `iisc-roadshow-2023.jpg` - IISc Bangalore Roadshow 2023 image

## Supported Formats:
- JPG/JPEG
- PNG
- WebP

## Recommended Image Specifications:
- **Aspect Ratio**: 2:1 (e.g., 600x300px)
- **Resolution**: 600x300px to 1200x600px
- **File Size**: Under 500KB for optimal loading
- **Quality**: High quality but web-optimized

## How to Add Your Images:
1. Save your expo photos with the exact filenames listed above
2. Copy them to this `images/expo/` directory
3. Refresh your website to see the changes

The website will automatically load these images in the Expo - Experience modal.
